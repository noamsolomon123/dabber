# דַּבֵּר

הקלדה קולית בעברית, חינמית ומקומית לחלוטין — 100% offline, מבוסס [whisper.cpp](https://github.com/ggerganov/whisper.cpp) ומודל עברית של [ivrit-ai](https://huggingface.co/ivrit-ai).

> כל העיבוד מתבצע על המכשיר עצמו. שום הקלטה, שום טקסט ושום נתון לא נשלחים לשרת חיצוני — אין צורך בחיבור לאינטרנט.

---

## התקנה

1. עברו לעמוד ה‑[Releases](../../releases) של הפרויקט והורידו את קובץ ה‑APK העדכני (`app-debug.apk`).
2. הפעילו במכשיר את האפשרות **התקנה ממקורות לא ידועים** (Install from unknown sources):
   - באנדרואיד 8 ומעלה ההרשאה ניתנת לכל אפליקציה בנפרד — בעת ההתקנה תתבקשו לאשר את היישום שדרכו פותח הקובץ (למשל Chrome או מנהל הקבצים).
   - ניתן גם להגדיר ידנית דרך **הגדרות → אפליקציות → גישה מיוחדת → התקנת אפליקציות לא ידועות**.
3. פתחו את קובץ ה‑APK שהורדתם ואשרו את ההתקנה.

> הערה: בשלב זה ה‑APK חתום במפתח ה‑debug האוטומטי של אנדרואיד. הוא ניתן להתקנה לחלוטין, אך עדיין אינו מהדורת חנות רשמית.

## הרשאות נדרשות

| הרשאה | לשם מה | איך מפעילים |
|-------|--------|-------------|
| **מיקרופון** (Microphone) | הקלטת הקול שלכם לצורך תמלול. ההקלטה נשארת על המכשיר בלבד. | בפעם הראשונה שתפעילו הקלטה תופיע בקשת הרשאה — אשרו אותה. ניתן לנהל ב**הגדרות → אפליקציות → דַּבֵּר → הרשאות → מיקרופון**. |
| **הצגה מעל אפליקציות אחרות** (Draw over other apps / Overlay) | הצגת בועת המיקרופון הצפה מעל כל אפליקציה, כדי שתוכלו להכתיב בכל מקום. | **הגדרות → אפליקציות → דַּבֵּר → הצגה מעל אפליקציות אחרות** והפעילו את המתג. |
| **נגישות** (Accessibility) | זיהוי שדה הטקסט שבפוקוס והזרקת הטקסט המתומלל ישירות לתוכו. | **הגדרות → נגישות → דַּבֵּר** והפעילו את השירות. |

> שירות הנגישות משמש אך ורק להכנסת הטקסט לשדה הפעיל — הוא אינו אוסף ואינו שולח מידע לשום מקום.

---

## איך זה עובד

1. **בועת מיקרופון צפה** מופיעה מעל המסך בכל אפליקציה.
2. לחיצה על הבועה מתחילה **הקלטה** של הקול שלכם.
3. ההקלטה מועברת למנוע **Whisper** הרץ על המכשיר (במצב עברית) ומתומללת לטקסט — ללא אינטרנט.
4. הטקסט עובר **ניקוי קל** (רווחים, פיסוק וסידור בסיסי).
5. הטקסט המוכן **מוזרק אוטומטית לשדה שבפוקוס** דרך שירות הנגישות.

הכול קורה מקומית על המכשיר. אין שרתים, אין מעקב, אין תלות ברשת.

---

## בנייה מהמקור

דרישות מוקדמות:

- **JDK 21**
- **Android SDK** (כולל NDK ו‑CMake, מותקנים אוטומטית על‑ידי Gradle)

בנייה:

```bash
./gradlew assembleDebug
```

ה‑APK ייווצר בנתיב:

```
app/build/outputs/apk/debug/app-debug.apk
```

> בנייה אוטומטית רצה גם ב‑GitHub Actions (ראו `.github/workflows/android.yml`); דחיפת תג בתבנית `v*` מצרפת את ה‑APK ל‑Release.
